Neratica.pl - Serwer MMO-RPG

🟢 Aktywny 01.2025 - OBECNIE Game Development

Innowacyjny serwer Minecraft z autorskim systemem RPG i ponad 50 customowymi modelami. Projekt łączy elementy klasycznego MMORPG z unikalnymi mechanikami i elementami survivalu.

  • Customowa mapa z pięknymi miastami i ciekawymi dunegonami
  • Autorskie pluginy
  • System miast i terytoriów PvP
  • Customowe przedmioty i craftowanie
  • System generowanie dungeonów

Technologie:

Java Spigot API MySQL WorldEdit Custom Plugins
